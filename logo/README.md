# Eden logo assets

The Eden wordmark and icon. The mark is the lowercase word **eden** — the leading
**e** in the brand blue, **den** in warm ink — set in Avenir Next Demi Bold (weight 600,
tracking −0.035em). The square **e** is pulled out on its own as the app icon / favicon.

These are finished assets only; nothing here is wired into the app.

## Colors

| Token | Hex | Use |
| --- | --- | --- |
| Brand blue (light) | `#4A7DFF` | the `e`, on light grounds — `--primary` |
| Brand blue (dark) | `#588AFF` | the `e`, on dark grounds — `--primary` (dark) |
| Ink | `#1A1A18` | `den`, on light grounds — `--eden-fg` |
| Off-white | `#FAF8F4` | `den`, on dark grounds — `--eden-band-fg` |

## Wordmark

| File | Notes |
